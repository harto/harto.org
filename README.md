# harto.org

My personal website, generated by [Jekyll][jekyll].

Requires:

 * [make][make]
 * [Bundler][bundler]
 * [npm][npm]


## Development

```
$ bundle install
$ bin/serve
```


## Deployment

```
$ bin/deploy
```

 [make]: http://www.gnu.org/software/make/
